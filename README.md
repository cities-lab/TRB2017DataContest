# TRB2017DataContest
TRB 2017 - Transportation Data Analytics Contest

## data folder
If you work on sapporo from rstudio web client, you can link to the shared data folder by running the following command in R console within RStudio:

`file.symlink("/home/TRB2017DataContest/data", "~/TRB2017DataContest/data")`

**DO NOT** check in the contents in the data folder onto github.
